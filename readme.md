### Fake News Detection Using Eye Tracking Data

This code applies machine learning algorithm (XGBoost) on a Fake News Dataset with Eye-tracking data.


#### Setup & Run

```

virtualenv venv

source venv/bin/activate

pip install -r requirements.txt

python app.py

```
